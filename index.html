Quais os comandos para se criar um site?

Aqui estão os principais comandos utilizados

h1 até h6: Títulos e subtítulos

p: Marcações de parágrafos

strong: Textos em negrito

html: Estruturar o conteúdo de uma página web

head: Contém metadados e links de recursos para a página web.

body: Define a parte visível da página web

title: Título da página

a: Liga os links com os conteúdos

ol: Cria uma lista ordenada

ul: Cria uma lista não ordenada

li: Itens de uma lista

img: Insere uma imagem na página web
